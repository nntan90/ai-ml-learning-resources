## 🐍 All Algorithms Implemented in Python

**Repo:** [TheAlgorithms/Python](https://github.com/TheAlgorithms/Python) | ⭐ 218,751  
**Mô tả:** Mọi thuật toán quan trọng được implement từ scratch bằng Python, có docstrings và test cases.

### Danh mục chính
- **Searches:** Linear, Binary, Jump, Interpolation, Fibonacci, Ternary
- **Sorting:** Bubble, Merge, Quick, Heap, Counting, Radix, Tim Sort
- **Data Structures:** Linked List, Stack, Queue, BST, AVL, Heap, Trie, Graph
- **Graph:** BFS, DFS, Dijkstra, Bellman-Ford, A*, Kruskal, Prim
- **Dynamic Programming:** Knapsack, LCS, LIS, Coin Change, Edit Distance
- **Math:** GCD, Sieve of Eratosthenes, FFT, Modular Exponentiation
- **ML:** Linear/Logistic Regression, K-Means, KNN, Neural Network (from scratch)
- **String:** KMP, Rabin-Karp, Levenshtein Distance
